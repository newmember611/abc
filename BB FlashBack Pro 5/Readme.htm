BB FlashBack Professional Edition Features

 

  • Captures all desktop activity to a movie file.
  • Export to Flash, QuickTime (H264), Windows Media Video (WMV), AVI, Powerpoint, GIF, MPEG4 and standalone EXE file formats.
  • Annotation of movies with sound and text.
  • High performance capture process is virtually undetectable on modern PCs.

System Requirements

BB FlashBack runs on Windows XP or newer. 32 and 64 bit OS versions are supported.


QuickTime7 or newer is required to export to QuickTime format.

Windows Media Player 9 or higher is required to export to Windows Media Video (WMV) format.

MS Office 2000 or newer, including PowerPoint, is required to export to MS PowerPoint format.

Flash exports that make use of integrated playback controls or FLV encoding require Flash Player version 9 or above to play.

 

 

 

Installation Requirements

BB FlashBack requires that installation be run by a user with administrator rights.
